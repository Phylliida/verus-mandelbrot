## Key Lessons Learned

1. **=~= scoping**: extensional equality facts inside `assert by` blocks don't leak out. Always assert =~= in the outer scope.

2. **Ghost snapshot timing**: capture output Seqs BEFORE vset (which modifies sign bytes) and AFTER exec helper (which modifies limb data). The two captures serve different purposes.

3. **Frame chain pattern**: capture `wg_mem_body_start` at loop body entry, assert pointwise frame `wg_mem[j] == wg_mem_body_start[j]` at end, using non-overlap arithmetic.

4. **nonlinear_arith transitivity**: Z3 can't derive `a*c < d` from `a < b` and `b*c < d`. Need explicit intermediate: `assert(a*c < b*c)` then `assert(a*c < d)`.

5. **by(nonlinear_arith) requires**: strips context — local let bindings must be included explicitly. Include the binding definition (e.g., `sum_u == z_re_u + z_im_u`).

6. **Zeroing loop invariants**: when zeroing multiple regions sequentially, each zeroing loop must carry frame conditions for previously-zeroed regions.

7. **History invariants**: for shared-buffer iteration patterns (write slot k, read slot k later), need forall invariant over all prior slots + frame postcondition on the helper function.

8. **Trigger matching**: `spec_orbit_no_escape`'s forall uses `#[trigger] spec_ref_orbit(z0, c_ref, k as nat)` where k: int. To instantiate with k: nat, create `let k_int: int = k as int; assert(spec_ref_orbit(z0, c_ref, k_int as nat) === z_k)`.

9. **Vote buffer initialization**: GPU shared memory is uninitialized. Any code that reads from shared memory slots written by other threads must ensure ALL slots are initialized — including slots for threads that may not participate.
